# ✨sayHello✨
- 技术栈:MongoDB,Express,React,Node.js,Socket.io
![Demo App](/frontend/public/readme_png.png)
### Setup .env file

```js
MONGODB_URI=...
PORT=5001
JWT_SECRET=...

CLOUDINARY_CLOUD_NAME=...
CLOUDINARY_API_KEY=...
CLOUDINARY_API_SECRET=...

NODE_ENV=development
```

### Build the app

```shell
npm run build
```

### Start the app

```shell
npm run start
```
